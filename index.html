what is it?

Responsive Web Design is an approach to display a website across multiple devices such as phones, tablets, and desktops.

HOW?

HOW?

HOLD FOR IMAGE OPACITY

Responsive design makes it possible for elements to adapt to different screen sizes, allowing the user optimal content display no matter what they view the website on.

This practice is used to make the best and most consistent experience possible for you, the user.

If a website is responsive, it will offer the same content and functionality across devices.
